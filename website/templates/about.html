About Toxic Terminator

This is a proof of concept demonstrating the functionality of Toxic Terminator.




ToxicTerminator was created by Jacques Fleischer, Heriberto Lopez, and Joshua Marks as a part of a COP3530 Data Structures and Algorithms final project at the University of Florida, go gata. The app is a full-stack application meant to serve as a proof-of-concept for pairing a naive word censor using comparable data structures, alongside real video game chats to quantify toxicity in online spaces.

The backend is written in C++ and the frontend is written in Python, a match made in limbo. The website is able to be locally hosted on your own computer, where you can add some games and datasets of your own for further analysis. If this is something that really interests you, then check out the GitHub repository at https://github.com/jpfleischer/ToxicTerminator
